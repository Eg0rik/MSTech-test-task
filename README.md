# MSTech-test-task
### Стэк:
- SwiftUI
- UserDefaults
- Combine
- архитектура MVVM

### Описание приложения:
1. При первом запуске показываем экран Онбординг
2. На главном экране показываем описание подписки (если пользователь выбрал подписку)
3. На экране выбора подписки показываем все возможные варианты подписки и при нажатии сохраняем в UserDefaults
